# 🚽 Toilet Bank System




## 🏦 프로젝트 소개
프로젝트 간단히 소개


1. **주제 - 은행 계좌 관리 시스템**: 다양한 은행에 있는 고객의 계좌를 조회하는 기능을 포함한 시스템입니다. 고객은 TOILET_BANK, KB_BANK, HANA_BANK 총 3개의 은행에 보유하고 있는 보통예금(입출금통장), 정기예금, 정기적금 총 3개 종류의 모든 계좌를 조회할 수 있습니다.
